Sophia Bae's Index

A collection of design and code explorations at the intersection of typography, history, and the web.

Scroll

These projects were created for Typography & Interaction — a year-long course in the Communication Design MPS program at Parsons, The New School. Each piece explores a different format of the web: from collaborative editorial layouts to game interface-like inventories.
